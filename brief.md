# Project Brief: E-Commerce Platform for Artisan Coffee

## Overview
We need a modern e-commerce platform to sell high-end artisan coffee beans and subscription boxes.
The target audience is coffee enthusiasts who value quality and sustainability.

## Key Features
- **User Accounts**: Login via email/social. Profile management.
- **Product Catalog**: Advanced filtering (roast level, origin, flavor profile).
- **Subscription Engine**: Users can build custom recurring orders (frequency, quantity).
- **Payment Gateway**: Integration with Stripe.
- **Admin Dashboard**: Inventory management, order fulfillment, sales analytics.
- **Blog/Content**: CMS sections for brewing guides and origin stories.

## Non-Functional Requirements
- **Performance**: Fast load times (<2s).
- **Security**: PCI compliance for payments. Secure user data.
- **Scalability**: Handle up to 10k daily visitors initially.
